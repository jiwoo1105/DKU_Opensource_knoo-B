## 프로젝트 소개
감정 분석을 통한 도서/영화 추천 시스템입니다. 사용자의 현재 감정 상태를 분석하여 적절한 도서와 영화를 추천해주는 서비스를 제공합니다.

## 주요 기능
1. 사용자 감정 분석
2. 도서/영화 메타데이터 기반 추천
3. 통합 검색 시스템 (도서, 영화, 제목 검색)
4. 시각화된 분석 결과 제공
